Festabstand
===========

Festabstände sind kleine, nicht variable Zwischenräume zwischen Zeichen.
Sie erhöhen die Lesbarkeit von Texten. Weiterhin werden die Zeichen die
ein Festabstand verbindet niemals getrennt. Somit wird verhindert, dass
``z. B.`` niemals durch einen Zeilenumbruch getrennt wird. Anwendung:
Abkürzungen, beim Datum, Paragraphenzeichen, Rechenzeichen, Einheiten,
etc.

Beispiele:

.. code-block:: rst

    v.l.n.r. = v.<FESTABSTAND>l.<FESTABSTAND>n.<FESTABSTAND>r.<FESTABSTAND>
    z.B. = z.<FESTABSTAND>B.
